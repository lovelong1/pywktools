# 基于python的工具包-Pywkcloud
基于python的工具包,云应用相关
## 获取代码
```shell
git clone https://gitee.com/lovelong1/pywktools
cd py-wk-tools-cloud
# 创建虚拟环境
python -m venv .
# liunx mac
source venv/bin/activate
# windows
venv/Scripts/activate.bat
# 编译项目
python3 setup.py build
```

## 打包
```shell
python3 setup.py sdist bdist_wheel
```
## 安装
```shell
pip install Pywkcloud-1.0.1-py3-none-any.whl
```
## 卸载安装
```shell
pip uninstall Pywkcloud-1.0.1-py3-none-any.whl
```
